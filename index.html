The Main Things You Need To Know About Website Coding

The First Three Things That You Need To Know Are: HTML Stands For Hypertext Markup Language, CSS Stands For Cascading Style Sheet, And JS Stands For Javascript.
The Use Of HTML Is To Add Text Buttons Anything You Can Think. There Are 6 Sizes Of Text, There h1, The Biggest, And h6, The Smallest. The Use CSS Is To Design The Website; for classes and # for id's and for tags, you write the tag on itself. After Every ., id or tag, you put a {. And Finally, Javascript Is Used To Code All The Functions.
